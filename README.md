# Demo-App


## INSTALLATION

###### 1- Install and run django
  ```
  cd ../demoapp_django
  
  pip install -r requirement.txt
  
  python manage.py makemigration device
  
  python manage.py migrate

  python manage.py loaddata users.json
  
  python manage.py runserver
  
  ```
###### 2- Install vue run vue
  ```
  cd ../demoapp_vue
  
  npm install
  
  npm run serve

  ```
